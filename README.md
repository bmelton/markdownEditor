# Markdown Editor
A markdown editor written using [Angular 2](http://angular.io).

## Quickstart
1. Clone repository and `cd` in to project directory.
1. Install dependencies.
    ```bash
    npm install
    ```
1. Run server.
    ```bash
    npm run go
    ```
1. Visit [http://localhost:3000](http://localhost:3000) to view the editor.
